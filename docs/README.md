# G7 Portfolio project notes

## Purpose
Preserve the G7 portfolio as a maintainable static site.

## Structure
- `index.html` for the public entry point
- `404.html` for the fallback page
- `robots.txt` and `sitemap.xml` for discoverability
- `vercel.json` for deployment settings

## Maintenance
- Keep source and content in the root site tree.
- Keep operational notes here in `docs/`.
